Lab 1 Grade

using TDD practices to implement a grade calculator application which calculates a student's final grade

Key Components:

public static GradecalculateGrade(int labPoints, float examGrade): Calculates the final grade.
Rules:

Lab points (50%) + Exam grades (50%).
Exceptions for out-of-range values.
Cum laude status for grades > 30.
Lab points influence exam grades (max bonus: 3 points).
Development Approach:

Follows TDD principles.
Each function limited to 10 lines.
Encourages code refactoring for cleanliness.
